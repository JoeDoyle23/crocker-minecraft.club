About the Crocker Minecraft Club

The Crocker Minecraft Club is a private Minecraft server where members can learn to play Minecraft in a safe and protected place. Our goal is to help foster proper online etiquette when playing with others. Parents can read more about the Crocker Minecraft Club on the For Parents section.

Please read the Server Rules with your child before playing.

Server Rules

In order to have an environment where everyone can have fun, there are a few simple rules for this server.

  1. Be nice to other players
  2. Do not destroy another player's property
  3. No stealing from other players

Players who don't follow the rules will lose access to the server.

Information for Parents

Minecraft can be a fun way for kids to start learning about computers. The hard part can be understanding how to start! This server is setup to allow the Crocker community to learn to play Minecraft with their friends. To help you understand how all this works, here are some answers to common questions.

  • What do we need to join the server?
    You will need to own a copy of Minecraft that runs on a PC, Mac, or Linux.
  • Can I play with Minecraft Pocket Edition? (On Android, iPhone, iPad, Xbox)
    Minecraft Pocket Edition is a different version of Minecraft designed for mobile devices. It's not actually the full version of Minecraft, and is unable to connect to this server.
  • How much does it cost to play?
    Playing on the server is free. You do need to own a copy of Minecraft. You can purchase it at minecraft.net
  • What is different when playing on a server compared to playing alone?
    This server is setup to be the same as when playing in Single Player, except players can work together. The Nether and The End dimensions are enabled (you might need to ask your kids to explain that to you!)
  • What is the game mode? Are there cheats enabled?
    This server is running in Survival Mode. There are no cheats enabled.
  • I like Minecraft! Can I play too?
    Sure! No reason parent's can't play too.
  • How do we get started?
    Initially, the server will be by invite only. After I get a feeling for how everything is working, we'll open it up to the broader Crocker Community.
