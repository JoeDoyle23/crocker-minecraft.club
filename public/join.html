Joining the Crocker Minecraft Club

The Crocker Minecraft Club is a private Minecraft server where members can learn to play Minecraft in a safe and protected place. Our goal is to help foster proper online etiquette when playing with others.

Please read the Server Rules with your child before joining.

To join, please send an email to joe@joedoyle.us with your child's name and Minecraft username. The server is restricted to only allow authorized players.

Connecting to the Crocker Minecraft Club

For the Minecraft experts out there, you can connect to the server by adding crocker.minecraft.club in the Multiplayer section of Minecraft.

For everyone else, you can follow the steps below.

1. At the main Minecraft menu, choose Multiplayer

2. Click the Add Server button

3. Enter the server name and address as seen below, then click Done

4. You should then see the server added to the list. You may need to click the Refresh button to update the status.

5. To join, click on the server and click Join Server
